## 基本的な使用シナリオ

### 朝のルーティン

**シナリオ:** 毎朝の習慣として、その日の計画と目標を整理

```discord
# #memo チャンネルに投稿
今日のタスク：
- [ ] プロジェクト A の進捗レビュー
- [ ] 顧客への提案書作成
- [ ] 夕方のチームミーティング準備
- [ ] 週次レポートの下書き

目標：
集中力を維持して、提案書を今日中に完成させる
```

**結果:**
- デイリーノートに自動統合
- タスクが個別に抽出・管理
- 進捗追跡が自動化
